  • What is grid based design?

A grid is a structure made up of a series of intersecting straight (vertical,horizontal, and angular) or curved guide lines used to structure content. The grid serves as an armature or framework on which a designer can organize graphic elements.

  • Whats all the hype about responsive webpages?

With the multitude of devices out there - desktops, laptops, phones, responsive design means developing websites which will resize and adapt to different sizes of screen (different resolutions) and types of devices, ensuring the website is still legible and is looking great.

  • What is semantic structure?

In HTML, tags are used to identity groups of content such as title, header, paragraphs, images. This allows for an easy to follow structure and shows the basic layout of the information.
